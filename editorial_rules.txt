# Автогенерация из editorial_rules.xlsx
# Не редактируйте вручную — редактируйте xlsx файл
#
# Форматы: GREP<Tab>паттерн<Tab>замена  |  найти<Tab>заменить
# топ 16 → Топ-16, ТОП-3 → Топ-3 (только ядро слова, окружающие кавычки/скобки не трогает)
GREP	[Тт][Оо][Пп]\s*[-–]?\s*(\d+)	Топ-$1
# лига чемпионов → Лига чемпионов (название турнира с заглавной)
лига чемпионов	Лига чемпионов
# чемпионат мира → Чемпионат мира (название турнира с заглавной)
чемпионат мира	Чемпионат мира
# чемпионат России → Чемпионат России (название турнира с заглавной)
чемпионат России	Чемпионат России
# первый канал → Первый канал (название СМИ с заглавной)
первый канал	Первый канал
# Ростех Арена → Ростех-Арена, ВТБ Арена → ВТБ-Арена. Минимум 3 буквы в имени, чтобы не ловить предлоги (На, В)
GREP	([А-ЯЁA-Z][А-Яа-яЁёA-Za-z]{2,})\s+([Аа]рен(?:а|ы|е|у|ой))	$1-$2
# КАМАЗ → КамАЗ (правильное написание бренда)
КАМАЗ	КамАЗ
# Многоточие внутрь кавычек только для прямой речи (3+ слов). «Спартак»... не трогает, «Я думаю, это правда»... → «Я думаю, это правда...»
GREP	«([^«»]*\s[^«»]*\s[^«»]*)»(\.{3}|…)	«$1$2»
# выстрелил → «выстрелил», стрелял → «стрелял», прострелил → «прострелил» (метафора в кавычки)
GREP	(?<![«а-яёА-ЯЁ])([Вв]ыстрел[а-яёА-ЯЁ]*|[Пп]рострел[а-яёА-ЯЁ]*|[Сс]треля[а-яёА-ЯЁ]*)(?![а-яёА-ЯЁ»])	«$1»
# снаряд → «снаряд», снарядом → «снарядом» (метафора в кавычки)
GREP	(?<![«а-яёА-ЯЁ])([Сс]наряд[а-яёА-ЯЁ]*)(?![а-яёА-ЯЁ»])	«$1»
# Исключение: футбольный «снаряд» → футбольный снаряд (не метафора, убираем кавычки)
GREP	((?:[Фф]утбольн|[Хх]оккейн|[Бб]аскетбольн|[Вв]олейбольн|[Гг]андбольн|[Тт]еннисн|[Бб]ейсбольн|[Сс]портивн)[а-яёА-ЯЁ]*)(\s+)«([Сс]наряд[а-яёА-ЯЁ]*)»	$1$2$3
# стандарт → «стандарт», со стандарта → со «стандарта». Только падежные формы существительного, не ловит «стандартный», «стандартизация»
GREP	(?<![«а-яёА-ЯЁ])([Сс]тандарт(?:а|у|ом|е|ы|ов|ам|ами|ах)?)(?![а-яёА-ЯЁ»])	«$1»
# Камаз → КамАЗ (дополнительный вариант написания)
Камаз	КамАЗ
